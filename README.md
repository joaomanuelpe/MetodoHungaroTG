# Método Húngaro (Hungarian Algorithm)

Este projeto implementa o **Método Húngaro (Hungarian Algorithm)**, um algoritmo clássico da área de **Teoria dos Grafos** e **Otimização**, utilizado para resolver o problema de atribuição de forma eficiente.

---

O objetivo do algoritmo é encontrar a melhor correspondência entre dois conjuntos (por exemplo, tarefas e agentes), **minimizando o custo total** ou **maximizando o lucro**, a partir de uma matriz de custos.

---

A implementação aborda todas as etapas fundamentais do método, incluindo:

- Redução de linhas e colunas da matriz  
- Cobertura de zeros com o menor número de linhas  
- Ajuste iterativo da matriz até a obtenção da solução ótima  

---

Este projeto foi desenvolvido como parte da disciplina de **Teoria dos Grafos**, com foco no entendimento prático do funcionamento do algoritmo e sua aplicação em problemas reais de **otimização combinatória**.
